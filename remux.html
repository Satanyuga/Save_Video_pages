what does remux do?
remux fixes any issues with the file container, such as missing time info. it helps increase compatibility with old software, such as vegas pro and windows media player.
how does it work?
remuxing takes existing codec data and copies it over to a new media container. it's lossless; media data doesn't get re-encoded.
on-device processing
cobalt remuxes files locally. files never leave your device, so processing is nearly instant.
